# 🕉️ Bhairavakona Gangamma Thalli Temple - Official Website

A responsive, accessible, and spiritually uplifting website for Bhairavakona Gangamma Thalli Temple, ready for GitHub Pages hosting.

## 🚀 Quick Start

### 1. Upload to GitHub
- Create a new public repository on GitHub
- Upload all files from this folder
- Go to **Settings → Pages**
- Select branch: `main`, folder: `/ (root)`
- Click **Save**

Your site will be live at: `https://yourusername.github.io/repo-name/`

### 2. Customize Content
Edit `index.html` to update:
- Temple name, address, contact details
- Darshan timings and event information
- Donation options and payment integration

### 3. Add Your Images
Place photos in the `images/` folder:
- `temple-hero.jpg` - Main hero background (1920x1080 recommended)
- `favicon.ico` - Browser tab icon (32x32px)
- Gallery images for the photo section

Update CSS in `css/style.css`:
```css
.hero {
  background: linear-gradient(...), url('../images/temple-hero.jpg') center/cover;
}
